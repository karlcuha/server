Проект реализует задание - человек отправляет ключевое слово на сервер, сервер возвращает список URL, клиент выбирает URL и скачивает контент через сервер. Загруженный контент сохраняется в "LocalStorage" и доступен для оффлайн-чтения.

 Стек

 Node.js 18+ версии
 Express
 Fetch API на клиенте
 LocalStorage для оффлайн-хранения
 Потоковая загрузка с отображением размера и прогресса

 Возможности
 Сервер хранит соответствие ключевых слов и URL.
 Клиент вводит ключевое слово и получает список URL без перезагрузки страницы.
 Клиент выбирает URL и скачивает контент через сервер.
 Во время загрузки отображаются размер и прогресс.
 Потом контент сохраняется в "LocalStorage".
 Сохранённый контент можно открыть оффлайн.
 Ошибки показываются пользователю в удобном виде.

 Структура проекта

```text

├── package.json
├── server.js
└── public
    ├── index.html
    ├── styles.css
    └── app.js
```

 Запуск локально

1 Установить Node.js версии 18 или выше.
2 Установить зависимости:

```bash
npm install
```

3 Запустить сервер

```bash
npm start
```

4 Открыть сервер в браузере

```text
http://localhost:3000
```

 Доступные ключевые слова

 "javascript"
 "nodejs"
 "html"
 "css"

Список ключевых слов и URL находится в файле "server.js" в объекте "keywordUrlMap".
