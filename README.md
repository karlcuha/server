В этом проекте человек отправляет ключевое слово на сервер, а сервер отвечает списком URL. Пользователь выбирает нужный URL, и контент скачивается через сервер. Загруженный материал сохраняется в LocalStorage, чтобы его можно было читать офлайн.

Технологии
Используется Node.js версии 18 и выше, Express для сервера, Fetch API на стороне клиента и LocalStorage для хранения данных без подключения к интернету. При скачивании контента показываются текущий размер и прогресс загрузки.

Что умеет проект
Сервер хранит связь между ключевыми словами и URL.
Пользователь вводит слово и без перезагрузки страницы получает список сайтов.
Он выбирает нужный URL и загружает контент через сервер.
Во время загрузки видны прогресс и объем данных.
После скачивания контент сохраняется в LocalStorage.
Сохраненную информацию можно открыть без интернета.
Ошибки выводятся пользователю в понятном виде.

Как запустить на компьютере

1. Установить Node.js версии 18 или выше.
2. В терминале установить зависимости командой

npm install

3. Запустить сервер командой

npm start

4. Открыть в браузере страницу

http://localhost:3000

Доступные ключевые слова

javascript
nodejs
html
css

Все ключевые слова и соответствующие URL хранятся в файле server.js в объекте keywordUrlMap.
